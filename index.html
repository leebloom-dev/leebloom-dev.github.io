HTML Attributes Summary

  • All HTML elements can have Attributes.
  • <a> has href attribute that specifies the URL of a page link.
  • <img> has src, path to the image to be displayed.
  • <img>: width, height - image size
  • <img>: alt - alternate text if image cant be dislayed.
  • styleattribute - adds styles to an element - css stuff
  • lang attribute in <html> - declares primary language of a website.
  • <title> attribute - extra text information when cursor hovers over something.
